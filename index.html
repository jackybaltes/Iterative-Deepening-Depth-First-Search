Implement a solution to the Jealous Husband problem, which is closely related to the missionaries and cannibal problem.

In the jealous husbands problem, there are three married heterosexual couples, that have to cross a river. There is a boat which can hold at most two people. No woman can be in the presence of another man unless her husband is also present.

Analysis of the Jealous Husband Problem

  • It is a good idea to estimate some of the parameters of a search problem to make sure that the search space is not too large
  • The most important parameters are the average branching factor b and the search depth d
  • Estimating the average branching factor
    • Initially, there are six people on the left side of the river
    • The boat holds at most two people
    • b = 6 (1 People) + 6*5 (2 People) = 36
  • Estimating the search depth of the puzzle
    • Minimum number of moves that are necessary to solve the puzzle
    • First two people from the left side to the right side, then one person back to left side
    • We can move 1 person from the left side to the right side with two steps (Except the last one)
    • 5*2 - 1, 9 moves
  • Estimated complexity Minimum number of nodes to expand: 387420489 nodes

def is_valid( state ):
    
    for w in [ W1, W2, W3 ]:
        if state[w] != state[w-1]:
            for h in [ H1, H2, H3 ]:
                if h != w - 1:
                    if state[h] == state[w]:
                        return False
    return True

def jealous_husbands_successors( state, hist = None ):
    children = []
    for p1 in range(1,NUM_OBJECTS):
        if state[p1] == state[BOAT]:
            ns = state.copy()
            ns[p1] = 1 - state[p1]
            ns[BOAT] = 1 - state[BOAT]
            if is_valid(ns):
                children.append( ns )
            for p2 in range( p1+1, NUM_OBJECTS ):
                if state[p1] == state[p2]:
                    ns = state.copy()
                    ns[p1] = 1 - state[p1]
                    ns[p2] = 1 - state[p2]
                    ns[BOAT] = 1 - state[BOAT]
                    if is_valid( ns ):
                        children.append(ns)
    return children

Implementation of Iterative Deepening Depth First Search

We will now implement the iterative deepening depth first search algorithm to solve the puzzle.

The algorithm maintains a cutoff value for the depth of the search.

If the algorithm fails to find a solution, then the depth limit is increased

def search( initial, successor_func, is_goal_func, dfs = True, node_limit = 1000, depth_limit = None ):
    queue = [ ( initial, [initial] ) ]
    num_nodes = 1
    max_queue = 0
    sol = None
    #print("Search started", queue)
    while( len( queue ) > 0 ):
        # current, chist = queue[-1]
        # queue = queue[0:-1]
        #print("Q", queue)
        if ( len(queue) > max_queue ):
              max_queue = len(queue)
        current, chist = queue.pop()
        #print("Num Nodes", numNodes)
        #print( len(chist), depthLimit, printState( current ) )
        if is_goal_func( current ):
            sol = chist
            break
        if ( depth_limit is None ) or ( len(chist) <= depth_limit ):
            #print("current state", current, chist )
            #print("queue", queue)
            children = successor_func( current )
            #print("children", children)
            for c in children:
                if c not in chist:
                    #print("Add child", printState(c))
                    if dfs:
                      queue.append((c, chist + [c] ))
                    else:
                      queue.insert(0, (c, chist + [c] ) )
                    num_nodes = num_nodes + 1
                    if ( num_nodes > node_limit ):
                        print("search terminated with num_nodes", num_nodes )
                        return ( num_nodes, max_queue, None )
        # else:
        #   print("Depth cutoff", len(chist), ">", depth_limit )
    return ( num_nodes, max_queue, sol )

def iddfs( initial, successors, is_goal, node_limit ):
    sol = (0,0,None)
    depth_limit = 1
    sumNodes = 0
    maxQueue = 0
    while( sol[2] is None ):
        sol = search( initial, successors, is_goal, True, node_limit, depth_limit )
        n,q,s = sol
        sumNodes = sumNodes + n 
        if maxQueue < q:
            maxQueue = q
        depth_limit = depth_limit + 1
    print("Depth limit", depth_limit )
    return ( sumNodes, maxQueue, sol[2] )
